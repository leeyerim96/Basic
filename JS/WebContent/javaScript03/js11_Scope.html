** 1. Variable 선언 Test **
=> 변수선언 : var(재선언 가능, 재할당가능), let(재선언 불가능, 재할당가능) 동일
=> 상수선언 : const(재선언 불가능, 재할당 불가능)
=> Type : 정수 ,실수, 문자열, boolean, null, undefined, function(), Object(내장객체 : Date...)
=> 전역 변수 선언시는 var 생략가능
=> 차이점 : var를 선언시에는 delete가 적용되지않고 변수와 같이 메모리에 남아있음.
