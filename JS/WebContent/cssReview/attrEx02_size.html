1. px: pixels (a dot on the computer screen) 크기, 위치

2. pt: point (1 pt = 1/72 inch) 글자크기

3. em: 배수, 1em 은 현재 폰트의 크기와 같습니다.
______ 2em 은 현재 폰트 크기의 2배 크기를 의미합니다.
______ 만약 현재 폰트의 크기가 12pt이면 2em은 24pt 크기에 해당합니다.

4. ex: 1ex 는 폰트의 높이(x-height)를 의미합니다.
